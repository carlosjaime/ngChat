# ngChat

A tiny-chat application with Angular, Socket.IO and NodeJs

## Requirements

- NodeJs

## Steps

###1 - Clone this repository

    git clone https://github.com/cyrilf/ngChat.git

###2 - Install the dependencies.

    [sudo] npm install

###3 - Open the browser

Enter this url in your web browser :

    localhost:2377

###4 - Enjoy and chat
